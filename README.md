# laravel-petstore-api

## Set Up.

```bash
$ composer install
```

```bash
$ npm install
```

## Start API Server.

```bash
$ php artisan serve
```

## Execute API.

```bash
$ curl -X GET http://localhost:8000/api/pets
```

## Run tests.

```bash
$ vendor/bin/phpunit
```

## Convert Open API Spec to JSON Schema.

```bash
$ npm run openapi2schema
```
